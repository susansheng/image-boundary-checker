## 🚀 下一步：部署到 Vercel

### 方式一：一键部署（推荐）

1. 点击下面的按钮直接部署：

   [![Deploy with Vercel](https://vercel.com/button)](https://vercel.com/new/clone?repository-url=https://github.com/susansheng/image-boundary-checker)

2. 使用 GitHub 账号登录 Vercel

3. 授权 Vercel 访问你的 GitHub 仓库

4. 点击 "Deploy"

5. 等待 2-3 分钟部署完成

6. 获取你的在线地址（类似：`https://image-boundary-checker.vercel.app`）

### 方式二：手动部署

1. **访问 Vercel**

   打开浏览器访问：https://vercel.com

2. **登录账号**

   使用 GitHub 账号登录

3. **导入项目**

   - 点击 "Add New..." → "Project"
   - 选择 `susansheng/image-boundary-checker` 仓库
   - 点击 "Import"

4. **配置项目**

   保持默认配置：
   - Framework Preset: `Other`
   - Root Directory: `./`
   - Build Command: （留空）
   - Output Directory: （留空）

5. **部署**

   点击 "Deploy" 按钮，等待部署完成

6. **访问应用**

   部署成功后，Vercel 会提供一个 URL
